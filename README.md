# ITransition Intern
## _Task 5_

Simple Contact Generation application using:

- React
- NodeJs
- MongoDB

### Web-Page

https://i-transition-task-5.vercel.app/

## Installation

Install the dependencies and devDependencies and start the server.

To launch client side
```sh
npm i
npm run start
```

To launch server side
```sh
cd server
npm i
npm start
```
